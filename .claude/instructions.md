# ЕЦОПУ — Единая цифровая образовательная платформа университета

## Дизайн
Макеты лежат в `docs/design/screenshots/`.
Перед реализацией любого экрана — ОБЯЗАТЕЛЬНО открой соответствующий
скриншот и следуй ему пиксель-в-пиксель.

## Дизайн-система
- Основной фон: #0B1437 (тёмно-синий)
- Карточки/панели: #111C44
- Акцентный цвет: #4318FF (синий/фиолетовый)
- Кнопки primary: #3B82F6
- Текст основной: #FFFFFF
- Текст вторичный: #A3AED0
- Бордер-радиус карточек: 16px
- Бордер-радиус кнопок: 12px
- Шрифт: Inter или DM Sans
- Sidebar: тёмный, ширина ~280px, с логотипом ЕЦОПУ

## Роли пользователей
1. **Студент** — мои проекты, активность, мероприятия, рейтинг, личный кабинет
2. **Преподаватель** — курируемые проекты, заявки студентов, календарь мероприятий, расписание, отчёты, активность, личный кабинет
3. **Заведующий кафедрой** — преподаватели, топ проекты, аналитика, расписание защит, отчёты, личный кабинет

## Стек
- Frontend: React + TypeScript + Tailwind CSS
- Backend: Node.js + Express (или FastAPI)
- БД: PostgreSQL
- Аутентификация: JWT + SSO

## Правила реализации
- Каждый экран реализуй как отдельную страницу/роут
- Sidebar — общий компонент, меню меняется по роли
- Все данные получай через API, используй моковые данные на старте
- Следуй макетам из docs/design/screenshots/
