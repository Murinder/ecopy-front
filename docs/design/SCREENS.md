### Левая панель — Форма входа (белый фон)

**Шапка формы:**
- Логотип: иконка книги (40×40px, синий квадрат с border-radius 12px) + текст "ЕЦОПУ" (18px, Bold)
- Подпись: "Единая цифровая образовательная платформа университета" (12px, серый)
- Отступ снизу: 32px

**Табы переключения:**
- Два таба: "Вход" (активный) | "Регистрация"
- Стиль: pill-кнопки в контейнере с border
- Активный таб: белый фон, тёмный текст, border
- Неактивный: прозрачный фон, серый текст
- Border-radius контейнера: full (pill)
- Margin-bottom: 32px

**Поле Email:**
- Label: "Email" (14px, Medium, тёмный)
- Input: height 48px, placeholder "student@university.edu"
- Border: 1.5px solid #E2E8F0, radius 12px
- Margin-bottom: 20px

**Поле Пароль:**
- Label: "Пароль" (14px, Medium, тёмный)
- Input: height 48px, type password, отображение точками
- Border: 1.5px solid #E2E8F0, radius 12px
- Margin-bottom: 16px

**Дополнительные элементы (строка):**
- Слева: чекбокс "Запомнить меня" (14px, Regular, серый)
- Справа: ссылка "Забыли пароль?" (14px, синий #3B82F6)
- Margin-bottom: 24px

**Кнопка "Войти":**
- Ширина: 100%
- Height: 48px
- Фон: #3B82F6, текст белый, 14px Medium
- Border-radius: 12px
- Margin-bottom: 24px

**Разделитель:**
- Горизонтальная линия с текстом "или" по центру
- Линия: 1px solid #E2E8F0
- Текст: 12px, серый, фон белый (перекрывает линию)
- Margin-bottom: 24px

**Кнопка "Войти через SSO":**
- Ширина: 100%
- Height: 48px
- Стиль: outlined (border 1.5px solid #E2E8F0, фон прозрачный)

**Карточка текущего проекта (выделенная, большая):**
- Фон: --bg-secondary с border

- Описание: текст описания проекта (14px, Regular)

- Метаданные в строку:
  - Иконка календаря + дедлайн ("15.12.2025")

- Hover: подсветка фона

**Формат:** выдвижная карточка или отдельная колонка (~380px)
